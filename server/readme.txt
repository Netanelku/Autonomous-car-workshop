<h1>Flask Application</h1>
    <p>This is a simple Flask application that demonstrates basic usage of Flask.</p>

    <h2>Description</h2>
    <p>This Flask application serves as a basic template for building web applications using Flask.</p>

    <h2>Usage</h2>
    <p>Clone this repository to your local machine:</p>

    <pre><code>git clone https://github.com/Netanelku/flask-application.git</code></pre>
    <p>Navigate to the project directory:</p>
    <pre><code>cd flask-application</code></pre>
    <p>Run the Flask application:</p>
    <pre><code>python app.py</code></pre>
    <p>Open a web browser and navigate to <a href="http://127.0.0.1:5000/">http://127.0.0.1:5000/</a> to view the application.</p>

    <h2>Routes</h2>
    <p><strong>/</strong>: Displays a simple message "Hello, World!".</p>
